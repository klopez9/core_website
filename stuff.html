Cool Stuff

This is a list of links to stuff that I think is pretty cool:

  • Codecademy - a great collection of web design tutorials which helped in the creation of this website
  • CoRE - the organization that hosts this website
  • The Scale of the Universe - an interactive Flash animation depicting different scales of the universe, from the cosmic to the microscopic
  • XKCD - an excellent webcomic focusing on math, physics, and CS humor
  • One Word - a great website if you have writer's block. How much can you write about a single word?
  • StumbleUpon - if you want to find more awesome websites or articles pertaining to your interests, visit this site
  • GitHub - the most commonly used website for computer scientists to work on group projects or publish their portfolios
  • One Chance - a darkly beautiful Flash game. What would you do if you had one chance to save the planet?
